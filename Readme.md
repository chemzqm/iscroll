# iscroll

Make element scroll on android < 4.

[demo](http://chemzqm.github.io/iscroll/)

## Installation

Install with [component(1)](http://component.io):

    $ component install chemzqm/iscroll

## API

```js
var iscroll = require('iscroll');
if (android_version < 4) {
  iscroll(el);
}
```
